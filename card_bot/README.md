# Создание карточек — Telegram Bot

Бот для создания визиток, пропусков и членских карт.

## Возможности
- регистрация пользователя (имя и возраст)
- профиль пользователя
- создание и сохранение шаблонов
- использование сохранённых шаблонов
- генерация карточки через Gemini-подобного AI помощника
- хранение данных в SQLite

## Установка
1. Установите Python 3.10+
2. Создайте виртуальное окружение
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скопируйте `.env.example` в `.env` и заполните:
   - `BOT_TOKEN`
   - `GOOGLE_API_KEY`
5. Запустите:
   ```bash
   python main.py
   ```

## Структура БД
- `users` — пользователи
- `templates` — шаблоны карточек
- `generated_cards` — сгенерированные карточки

## Замечание
AI-генерация работает через `google-genai`. Если ключ не указан, бот использует встроенный fallback и всё равно создаёт карточку.
